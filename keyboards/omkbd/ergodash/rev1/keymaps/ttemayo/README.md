# ttemayo's Keymap for [omkbd/ErgoDash](https://github.com/omkbd/ErgoDash)

# Overview
### [VIM-like functionality](../../../../../../users/ttemayo/readme.md#vimdows)
- basic VIM-like simulation outside of a VIM environment
### [Physical Key Tracking](../../../../../../users/ttemayo/readme.md#physical-key-tracking)
- Allows binding of different chords to macros
### [Multi-function Leader Key](../../../../../../users/ttemayo/readme.md#leader-key-and-macros)
- Single tap to trigger the Leader Key functionality
- Double tap to toggle CapsLock
- encase cursor with various symbols i.e. (), [], {}, "", etc., outside of your IDE
### [Caps Word](https://github.com/qmk/qmk_firmware/blob/master/docs/feature_caps_word.md)
- Double Shift = Caps Word
- BUG - registering any shift results in ending of Caps Word
### [Tri-state Layer](https://github.com/qmk/qmk_firmware/blob/master/docs/ref_functions.md#update_tri_layer_statestate-x-y-z)
- Symbol Layer access while VIM and TRNS are toggled

## FLASH Instructions
1. Build the firmware using QMK MSYS: `qmk compile` or `make --jobs=1 omkbd/ergodash/rev1:ttemayo`
   - On Pro-Micro: Approx. firmware size: 25500/28672 (88%, 3172 bytes free)
2. Flash firmware
   - Open QMK Toolbox App
   - Browse for *.hex file
   - Unplug TRRS connector, and plug in one half of keyboard via USB
   - Check QMK Toolbox: *** DFU device connected.
   - Check [ ] Auto-flash and hit the physical reset button... or hit the Flash button on the toolbox app
   - Repeat for second half of keyboard
   - Unplug USB, connect two halves with the TRRS cable, and reconnect to computer
